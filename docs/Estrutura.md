# Landing Page Profissional — Ryan Adrian

## 1. Objetivo da Página

A landing page deve funcionar como **uma vitrine profissional**, mostrando:

- Quem você é

- O que você sabe fazer

- Como você resolve problemas com tecnologia

- Como entrar em contato com você


O foco não é apenas listar tecnologias, mas **transmitir confiança, profissionalismo e clareza técnica**.

A pessoa que visitar a página precisa entender rapidamente:

> “Esse cara sabe o que está fazendo e pode resolver meu problema.”

---

# 2. Identidade Visual da Página

## Conceito Visual

A ideia principal é um **tema espacial minimalista**, representando:

- exploração

- tecnologia

- profundidade de conhecimento

- pensamento estruturado


### Fundo

- **cor base:** preto profundo `#0a0a0a`

- **efeito:** céu estrelado

- pequenas partículas brancas simulando estrelas

- estrelas com **opacidade variando entre 0.2 e 0.6**

- movimento lento e suave


Possíveis efeitos:

- **parallax leve**

- movimento horizontal lento

- partículas piscando lentamente


Inspiração visual:

- espaço

- minimalismo tecnológico

- interfaces futuristas


---

## Cards da Interface

Todos os conteúdos devem estar em **cards translúcidos**.

Características:

```
background: rgba(60,60,60,0.15)
backdrop-filter: blur(10px)
border: 1px solid rgba(255,255,255,0.05)
border-radius: 12px
```

Objetivo:

- manter contraste com o fundo

- criar sensação de profundidade

- manter a leitura confortável

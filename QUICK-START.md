## 🚀 DEPLOY EM 3 PASSOS

### 1️⃣ CRIE O REPOSITÓRIO
1. Acesse: https://github.com/new
2. Nome: `RyanLandPage`
3. Public ✓
4. Create repository

### 2️⃣ EXECUTE OS COMANDOS
Abra o PowerShell na pasta do projeto:

```bash
cd C:\GitHub\RyanLandPage

git init
git add .
git commit -m "🚀 Deploy inicial"
git remote add origin https://github.com/1Kibe/RyanLandPage.git
git branch -M main
git push -u origin main
```

### 3️⃣ ATIVE O GITHUB PAGES
1. Settings → Pages
2. Branch: `main`
3. Folder: `/ (root)`
4. Save

**✨ PRONTO!** Seu site estará em:
**https://1Kibe.github.io/RyanLandPage**
